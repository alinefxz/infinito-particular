# Infinito Particular
Este é um projeto em trio feito por Aline, Isadora e Lívia, onde compartilhamos tudo o que gostamos, como livros, filmes, nostalgia e muito mais.

# Objetivos
Criar um espaço para reunir nossos interesses e falar sobre o que nos inspira e diverte. Queremos que seja algo leve e variado, para explorar diferentes temas do nosso cotidiano.
